Findings

  • docs/changelog.md: Builds into _site/docs/changelog.md instead of HTML, leaving https://scripterix.github.io/docs/changelog/ as a 404. The file requests a changelog layout that does not exist in _layouts; switching to an existing layout (for example page) or adding the missing layout should unblock generation.
  • docs/docs.md: Text displays mojibake (� characters) indicating an encoding mismatch. Source likely saved in Windows-1250 while the site expects UTF-8; resave with UTF-8 to restore Polish characters.
  • Jekyll build warning: jekyll build reports “Pagination is enabled, but I couldn’t find an index.html page to use as the pagination template. Skipping pagination.” Confirm whether pagination is still required or disable it in _config.yml.

Next Steps

  1. Decide on the desired layout for the changelog page and ensure the layout file exists so GitHub Pages emits HTML.
  2. Normalise docs/docs.md encoding (UTF-8) and proof-read once characters render correctly.
  3. Either supply an index.html (or index.md) for pagination or remove the paginate settings to silence the warning.
