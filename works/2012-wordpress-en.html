Project details

This throwback project shows the ship cadence I used in 2012. The theme started as a static Figma mock, then was ported into modular sections inside the WordPress Customizer so the client could edit copy without touching code.

The scope covered newsletter tooling, lead forms, and a very light automation layer for follow-up emails. The big win: keeping load times under one second by compressing images and leaning on server-side caching.
