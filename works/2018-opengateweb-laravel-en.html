Project details

This project started as a whiteboard exercise to map reusable sections for OpenGateWeb’s sales site. I translated those sketches into Blade components backed by a tiny CMS so the marketing team could show or hide blocks without code changes.

Because the site runs on a shared host, I trimmed assets aggressively, introduced HTTP caching headers, and made sure Time to First Byte stayed below one second even when the longer case-study modules were active.
