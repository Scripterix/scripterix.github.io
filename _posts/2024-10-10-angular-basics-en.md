## 🅰️ Angular Fundamentals + Practice (My Path)

I've been building with Angular since version **1.0**. Today I ship applications in **Angular 18**, which unlocks a whole new level for modern SPAs.

### 📚 From Courses to Production Projects

My Angular story started back when **AngularJS** dominated. I gathered the most mileage on versions **8–14**, and a turning point was finishing the 🎓 **"Angular Front To Back"** course by **Traversy Media**.

That curriculum helped me:

- 🔧 set up my first development environment
- 💡 learn how to build components and services
- 🧑‍💻 ship a client management app powered by Firebase

### 🚀 Angular 18 — A New Chapter

I keep up with every Angular release. Right now I'm focused on **Angular 18**, which introduces game-changing features:

- 🧠 **Angular Signals** — reactive state management without leaning on RxJS
- 🧩 **Standalone Components** — no more module declarations
- ⚡ **Zoneless Change Detection** *(experimental)* — performance gains for interactive UIs
- 🎯 Streamlined ergonomics for smaller apps

Keeping my knowledge current lets me build **efficient, modular, and modern applications**.

### 🧑‍🏫 What a Solid Angular Course Teaches You

Here’s the progression I recommend (and followed myself):

- ❓ What is Angular?
- 🛠️ Environment setup: Node.js, VS Code, Git Bash
- 🧱 Angular CLI commands and configuration
- 🗂️ Project structure and dependencies
- 🧮 TypeScript fundamentals
- 🧩 Components and the `@Component` decorator
- 📦 Modules and `NgModule`
- 🔁 Data binding and interpolations like `title="{{ value }}"`
- 🔡 Types and interfaces
- 🧾 Templates (`ngIf`, `ngFor`)
- 🎨 `ngClass`, `ngStyle`, and Pipes
- 📝 Forms, events, `ngModel`
- 🔌 Services and dependency injection
- 🌐 Observables and HTTP clients
- 🧭 Angular routing
- ☁️ Firebase, Firestore, AngularFire
- 💬 Flash messages and UX flourishes

### 🚀 Course Takeaways

The course is a well-paced deep dive — from the absolute basics to Firebase integrations. We covered:
