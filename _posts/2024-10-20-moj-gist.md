Krótki przegląd mojego Gist na GitHub oraz przypomnienie podstawowych poleceń Git i Bash.

Gist Scripterix to zbiór krótkich skryptów i fragmentów kodu udostępnionych na GitHub. Znajdziesz tu przykłady w takich językach jak Python, Java, JavaScript oraz narzędzia przydatne podczas pracy z projektami. Repozytorium jest regularnie uzupełniane o nowe rozwiązania i techniki.

Spis przykładów i fragmentów (wybrane):

- ES6 / EcmaScript
- Bulma Carousel with Text Content
- Flex CSS
- Flex and Grid (grid nested in flex)
- Bootstrap Modal v4.5.3
- JWT
- Git / Git Bash przykłady i polecenia CLI

Przykładowe polecenia Git (szybkie przypomnienie):

```bash
# Inicjalizacja lokalnego repozytorium
git init

# Dodanie pliku do indeksu
git add <plik>
# Dodaj wszystkie pliki (obecny katalog)
git add .

# Usuń plik z indeksu
git rm --cached index.html

# Sprawdź status drzewa roboczego
git status

# Zatwierdź zmiany
git commit -m "Opis zmian"

# Wypchnij na zdalne repo
git push
```
